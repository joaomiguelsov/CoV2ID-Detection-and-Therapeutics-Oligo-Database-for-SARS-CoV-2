# CoV2ID-Detection-and-Therapeutics-Oligo-Database-for-SARS-CoV-2
Here we describe the CoV2ID, a free database built to facilitate the evaluation of molecular methods for detection of SARS-CoV-2 and treatment of COVID-19. The database evaluates the available oligonucleotide sequences (PCR primers, RT-qPCR probes, etc.)
